{% trans "Formatting Text" %}

{% blocktrans %}We use a simple syntax for formatting documents, blog posts, and events. It should only take a few minutes to pick it up. The easiest way to learn is by example. {% endblocktrans %}

{% trans "Text" %}

{% formattingexample %} {% blocktrans %} Use asterisks to make words **bold** or *italic*. Separate paragraphs with empty lines "Smart Quotes" and 'other characters' will automatically be--inserted... {% endblocktrans %} {% endformattingexample %}

{% trans "Headings" %}

{% formattingexample %} {% blocktrans %} # Use the hash symbol to create headings ## Two hashes creates a subheading {% endblocktrans %} {% endformattingexample %}

{% trans "Lists" %}

{% formattingexample %} {% blocktrans %} Use bullet points for lists: * Apple * Banana * Orange Dashes also work: - Juice - Coffee - Water {% endblocktrans %} {% endformattingexample %}

{% trans "Ordered Lists" %}

{% formattingexample %} {% blocktrans %} Numbers can be used for ordered lists: 1. One 2. Two 3. Three You can use any number for ordered lists. This makes for easy re-ordering: 1. One 1. Two 1. Three {% endblocktrans %} {% endformattingexample %}

{% trans "Links" %}

{% formattingexample %} {% blocktrans %} URLs will be automatically converted into links: http://democracynow.org/. You can also set the link text: [Get your news here](http://democracynow.org/). {% endblocktrans %} {% endformattingexample %}

{% trans "Quotes" %}

{% formattingexample %} {% blocktrans %} Use the ">" char for quotations: > If there is no struggle there is no progress. > > - Frederick Douglass You can nest quotes as well: > Hi, how are you doing? >> I'm great, how are you? > Hunky Dory! {% endblocktrans %} {% endformattingexample %}
